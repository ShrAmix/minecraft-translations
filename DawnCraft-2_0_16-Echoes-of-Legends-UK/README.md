## 📥 Встановлення

1. Скачай останній реліз з вкладки [Releases](../../releases)
2. Відкрий папку зборки в CurseForge:
   `Профіль → Відкрити папку (Open Folder)`
3. Скопіюй файли перекладу з заміною
4. Запускай гру
