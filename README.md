## 🔧 Як встановити переклад

1. Скачай папку потрібної зборки
2. Відкрий папку своєї зборки у CurseForge/Prism/MultiMC
3. Скопіюй файли перекладу з заміною оригінальних
4. Запусти гру і зайди в **Options → Resource Packs** — активуй ресурспак з українським перекладом (перетягни вліво або натисни стрілку), зазвичай він має в назві "UK"
5. Встанови мову гри на **Українська** (Options → Language)
6. Запускай гру — текст буде українською
