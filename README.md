## 🔧 Як встановити переклад

1. Скачай папку потрібної зборки
2. Відкрий папку своєї зборки у CurseForge/Prism/MultiMC
3. Скопіюй файли перекладу з заміною оригінальних
4. Запускай гру — текст буде українською

> Детальні інструкції є у папці кожної зборки в окремому `INSTALL.md`
